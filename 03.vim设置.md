# VIM设置

```text
set number
set showmode
set showcmd
set encoding=utf-8
filetype on
filetype indent on
filetype plugin on
syntax on

set tabstop=2
set shiftwidth=2
set expandtab
set softtabstop=2
set foldenable
set fdm=syntax

set wrap
set linebreak
set laststatus=2
set ruler
set showmatch
set hlsearch
set incsearch
set ignorecase
colorscheme desert
```
